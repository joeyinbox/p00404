What is a Logic Gate?

Logic gates are fundamental building block of digital circuit which control the flow of electricity through circuits. They are based on Boolean algebra logic. The computer systems have thousands of these logic gates constructed together which help them to processes different tasks logically. The data is converted to binary form (1s and 0s) in computers which is then process logically according to the logic gates arrangement. The logic gates make use of Boolean algebra operations such as AND, OR and NOT to control the flow of electricity through circuits. There are basic seven types of logic gates:

NOT Gate

NOT gate make use of NOT (negation) operator of Boolean algebra. It takes single true or false value and the result is always the opposite of the input according to Boolean logic of NOT. So it basically inverts the input value. The symbol for NOT gate is normally given as below. It is written as ~A..

Algebraic notation

(~A)

AND Gate

AND gate make use of AND (Conjunction) operator of Boolean algebra. It can take two inputs true or false values and output the correspondence result based on the Boolean logic. The symbol for AND gate is usually as below. It is written as A & B.

Algebraic notation

(A & B)

NAND Gate

NAND gate is the inverse of AND gate. It can tak two input 0 or 1 value and out the correspondence result based on the logic which is normally the opposite of the AND gate. It is written as A|B. The symbol is normally as below. The little circle in the output represent that it is inverse of AND..

Algebraic notation

(A | B)

OR Gate

OR gate make use of OR (disjunction) operator of Boolean algebra. It can also take two inputs true or false values as (1s and 0s) and output the correspondence result based on the Boolean logic. The symbol for OR gate is normally given as below. It is written as A+B.

Algebraic notation

(A + B)

NOR Gate

Similarly NOR gate is the inverse of OR gate. It can take two input values as 0s and 1s and the result is always the opposite than the OR gate. It is written as A-B. the symbol is normally as below..

Algebraic notation

(A - B)

XOR Gate

The XOR gate called exclusive OR is like an OR gate but only gives a true output if either a or b has true input. If both have true input the result c will be false. If both have false input the result again will be false. It is written as A(+)B. The symbol is normally as given below..

Algebraic notation

A (+) B

XNOR Gate

XNOR gate is inverter of XOR gate as it has a circle in the symbol. It can take two inputs and the result is always the opposite of XOR gate. It is written as A(.)B. the symbol is normally as given below..

Algebraic notation

A (.) B
